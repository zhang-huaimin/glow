Install ``glow``
----------------------------------------

``glow`` requires: Python 3.11+.

1. 安装命令:

.. code-block:: bash

    pip3 install .
    # or
    uv pip install .

2. Check that you installed the correct version:

.. code-block:: bash

    $ glow --version
    glow 0.0.1
